Class IA — Na⁺ Channel Block + ↑ APD (Prolongs QT → Torsades Risk)

Procainamide · Disopyramide · Quinidine (historical) Class IA — Intermediate Na⁺ Block + K⁺ Block Procainamide: IV or PO · Disopyramide: PO · Monitor QT interval
⚙ MOA
Intermediate Na⁺ channel kinetics + K⁺ channel (IKr) block:
• ↓ Vmax (phase 0) → ↓ conduction velocity → ↑ PR, ↑ QRS
• K⁺ block → ↑ APD → ↑ ERP → ↑ QT interval

Act on both atrial and ventricular tissue. Active metabolite of procainamide: NAPA.
⚡ Side Effects & Contraindications
QT prolongation → Torsades de Pointes (all Class IA)
Procainamide: Drug-Induced Lupus (anti-histone Ab+, anti-dsDNA−, complement normal)
Disopyramide: strong anticholinergic (urinary retention, dry mouth, constipation) + strong negative inotrope → avoid in HF

Contraindicated: Long QT syndrome, concurrent QT-prolonging drugs
🎯 Clinical Uses
Ventricular arrhythmias (VT/VF)AF conversionWPW + Afib: procainamide ★

Procainamide IV: VT when lidocaine ineffective.
WPW + Afib: use procainamide (blocks accessory pathway).
⭐ High-Yield Facts (Exam Focus)
All Class IA prolong QT → Torsades de Pointes risk (especially with hypokalemia/hypomagnesemia).
ECG changes: ↑ PR, ↑ QRS, ↑ QT.
Procainamide: drug-induced lupus — anti-histone Ab (+), anti-dsDNA (−), complement NORMAL. Stops when drug stopped.
Disopyramide: strong negative inotrope (avoid in HF) + strong anticholinergic (urinary retention).
WPW + Afib: use PROCAINAMIDE (blocks accessory pathway). Never adenosine/digoxin/verapamil.
Drug-induced lupus: anti-histone+, anti-dsDNA− (idiopathic SLE is anti-dsDNA+). Torsades risk highest when K⁺ or Mg²⁺ is low. Disopyramide in HF = dangerous (negative inotrope).

Class IB — Na⁺ Channel Block + ↓ APD (Use-Dependent — Ischemic Tissue)

Lidocaine (IV only) · Mexiletine (PO) Class IB — Fast Na⁺ Block — Use-Dependent Lidocaine: IV ONLY (high first-pass) · Mexiletine: PO · Monitor CNS symptoms
⚙ MOA
FAST Na⁺ channel kinetics (unbinds <1 sec) → preferentially bind inactivated channels → most effective in rapidly firing ischemic/tachycardic tissue.

↓ APD (shortens action potential duration) — opposite of Class IA.
USE-DEPENDENT: more effective at faster rates → ideal for ischemic VT.
⚡ Side Effects & Contraindications
CNS toxicity (dose-dependent): dizziness → tremor → slurred speech → SEIZURES → respiratory depression
Minimal ECG changes (minimal QRS/QT widening) — distinguishing feature from IA/IC
Mexiletine: GI upset, same CNS ADRs

Lidocaine INEFFECTIVE orally (extensive first-pass metabolism)
INEFFECTIVE for atrial arrhythmias
🎯 Clinical Uses
Ventricular arrhythmias (VT/VF) — post-MI ★Ischemic VT (use-dependent)Local anesthesia (lidocaine)

Mexiletine: oral IB agent — chronic ventricular arrhythmia prevention.
NOT for atrial arrhythmias.
⭐ High-Yield Facts (Exam Focus)
FAST kinetics → use-dependent → most effective in ischemic/tachycardic tissue (post-MI VT).
IB drugs SHORTEN APD → minimal/no QT prolongation (vs. IA/III which lengthen QT).
Minimal ECG changes — key distinguishing feature of Class IB from IA and IC.
Lidocaine: IV ONLY (extensive first-pass metabolism → oral = useless).
INEFFECTIVE for atrial arrhythmias — ventricular only.
CNS toxicity is dose-dependent: dizziness → seizures → respiratory arrest.
'Give oral lidocaine' → impossible (first-pass destruction). IB drugs USELESS in Afib/AFL. Minimal ECG change = IB hallmark. Contrast: Class IA ↑ QT; Class IB ↓ APD (minimal QT change).

Class IC — Marked Na⁺ Block (↑ QRS, ↑ PR — CAST Trial Warning)

Flecainide · Propafenone (+ β-blocking activity) Class IC — Slow/Marked Na⁺ Block PO · Monitor ECG (QRS, PR) · CONTRAINDICATED post-MI / structural heart disease
⚙ MOA
SLOW Na⁺ channel kinetics (unbinds >10 sec) → marked ↓ Vmax → ↑ PR interval + ↑ QRS duration.

Minimal effect on APD/QT (distinguishes from IA and III).
Propafenone: also has intrinsic β-blocking activity.
QRS prolongation is rate-dependent (greater at faster rates).
⚡ Side Effects & Contraindications
PROARRHYTHMIC — marked risk → can convert organized arrhythmia to VF
CONTRAINDICATED post-MI and structural heart disease (CAST trial — ↑ sudden death)
Flecainide: CNS toxicity, blurry vision, altered taste
Propafenone: β-blocker ADRs (bradycardia, bronchospasm)

Contraindicated: Post-MI, HFrEF, structural heart disease, bundle branch block
🎯 Clinical Uses
Paroxysmal AF — 'pill-in-pocket' ★ (normal heart ONLY)SVT

ONLY in structurally normal hearts.
NEVER in CAD, HF, LVH, prior MI.
⭐ High-Yield Facts (Exam Focus)
CAST TRIAL: flecainide/encainide suppressed PVCs post-MI but INCREASED sudden death 2.5×. Never use post-MI.
ECG changes: ↑ QRS (most prominent), ↑ PR; minimal QT change.
'Pill-in-pocket' for paroxysmal Afib: patient takes dose at symptom onset — ONLY if structurally normal heart.
Propafenone: also has β-blocking activity → watch for bradycardia/bronchospasm.
IC drugs suppress PVCs but INCREASE death in post-MI patients (CAST trial). IC drugs = safe ONLY in structurally normal hearts. Remember: IC suppresses PVCs but is lethal in structural disease.

Class III — K⁺ Channel Block (↑ APD, ↑ QT)

Amiodarone ★ (Class I+II+III+IV — 'Dirty Drug') Multi-Class Antiarrhythmic — Primary: K⁺ Block IV or PO (poor PO bioavailability) · t½ = 40–55 DAYS · Monitor TFTs, LFTs, PFTs, CXR
⚙ MOA
Blocks K⁺ channels (primarily) → ↑ APD + ↑ ERP + ↑ QT.
Also blocks: Na⁺ + Ca²⁺ channels + α/β receptors (Class I+II+III+IV properties).

37% iodine by weight → thyroid effects.
Highly lipophilic → accumulates in tissues (lungs, liver, thyroid, cornea, skin) → t½ 40–55 days → toxicity persists months after stopping.
⚡ Side Effects & Contraindications
Pulmonary toxicity: interstitial pneumonitis/fibrosis — most SERIOUS, potentially FATAL
Thyroid: hypothyroidism OR hyperthyroidism (iodine load → both directions — Jod-Basedow)
Hepatotoxicity (↑ LFTs — monitor regularly)
Corneal microdeposits (almost universal, usually asymptomatic)
Blue-gray skin (sun-exposed areas), photosensitivity

Drug interactions:
• CYP2C9 inhibitor → ↑ warfarin (↑ INR) → reduce warfarin dose
• P-gp inhibitor → ↑ digoxin levels
• Interactions persist months after stopping (long t½)
🎯 Clinical Uses
Pulseless VT/VF — ACLS 1st line ★AF/AFL (rate + rhythm control)HFrEF with arrhythmia (tolerated)Hemodynamically unstable VT

PARADOX: despite most QT prolongation, causes LESS torsades than other Class III.
⭐ High-Yield Facts (Exam Focus)
First-line for PULSELESS VT/VF in ACLS (300 mg IV → 150 mg).
t½ = 40–55 DAYS — drug and interactions persist MONTHS after stopping.
PARADOX: causes LESS torsades than other Class III despite most QT prolongation.
Thyroid toxicity goes BOTH ways: hypo (iodine load) OR hyper (Jod-Basedow). Check TSH regularly.
Amiodarone + warfarin → ↑ INR (CYP2C9 inhibition) — classic interaction. Reduce warfarin dose.
Ground-glass opacities + dyspnea + amiodarone → pulmonary toxicity. Get CXR + PFTs.
Blue-gray skin + antiarrhythmic = amiodarone (pathognomonic).
Blue-gray skin = amiodarone. Amiodarone stopped → toxicity persists MONTHS (t½ 40–55 days). TSH checks regularly — can go EITHER direction. CYP2C9 inhibitor → reduce warfarin dose when starting.
Dronedarone · Ibutilide (IV) · Dofetilide · Sotalol (+ Class II β-block) Class III — Other K⁺ Channel Blockers Ibutilide: IV only · Dofetilide: PO (renal dosing critical) · Sotalol: PO (renal dosing)
⚙ MOA
All block K⁺ channels (IKr) → ↑ APD → ↑ ERP → ↑ QT → Torsades risk.

Dronedarone: non-iodinated amiodarone analog → no thyroid effects; CYP3A4/2D6/P-gp inhibitor.
Sotalol: also Class II (β-blocker activity) → ↓ HR + ↓ AV conduction + K⁺ block.
Dofetilide/Ibutilide: more atrial-selective K⁺ block.
⚡ Side Effects & Contraindications
QT prolongation → Torsades de Pointes (all Class III)
Dronedarone: CONTRAINDICATED in CHF or permanent Afib (ANDROMEDA trial — ↑ mortality)
Sotalol: β-blocker ADRs (bradycardia, bronchospasm) + QT prolongation; renally cleared
Dofetilide: QT prolongation — requires in-hospital initiation; renally cleared (dose-adjust critical)
Ibutilide: torsades ~6% — have defibrillator available
🎯 Clinical Uses
Dronedarone: paroxysmal/persistent Afib (NOT CHF)Ibutilide: acute AF/AFL cardioversion (IV)Dofetilide: AF/AFL conversion + maintenanceSotalol: VT/VF + Afib maintenance
⭐ High-Yield Facts (Exam Focus)
Dronedarone: no iodine → no thyroid toxicity. CONTRAINDICATED in CHF and permanent Afib (ANDROMEDA: ↑ mortality).
Dofetilide: requires in-hospital initiation (monitor QTc for torsades). Critical renal dosing.
Ibutilide: IV cardioversion of Afib/AFL. ~6% torsades risk — defibrillator must be available.
Sotalol = Class II + Class III (β-blocker + K⁺ block) — two effects, two ADR profiles.
Dronedarone in CHF or permanent Afib = CONTRAINDICATED (↑ mortality — ANDROMEDA trial). Dofetilide requires in-hospital initiation. All Class III → QT prolongation → torsades risk.

Class II (β-Blockers) · Class IV (Non-DHP CCBs) · Other Agents

Metoprolol · Esmolol · Propranolol [Class II] · Verapamil · Diltiazem [Class IV] Class II: β-Blockers · Class IV: Non-DHP CCBs Class II: PO or IV (esmolol IV only) · Class IV: PO or IV · NEVER combine Class II + Class IV
⚙ MOA
Class II (β-blockers): β₁ blockade → ↓ cAMP → ↓ SA node automaticity + ↓ AV conduction → ↓ HR.

Class IV (non-DHP CCBs): L-type Ca²⁺ block in nodal tissue → ↓ SA + AV node activity → ↑ PR interval.

Esmolol: t½ ~9 min (erythrocyte esterase) → IV only, rapidly titratable rate control.
⚡ Side Effects & Contraindications
Class II: bradycardia, AV block, bronchospasm, mask hypoglycemia
Class IV: bradycardia, AV block, negative inotropy (↑ HF risk), ↑ digoxin (verapamil)
Constipation (verapamil)

NEVER combine Class II + Class IV → complete AV block/arrest
Non-DHP CCBs in HFrEF → worsen EF
Non-DHP in WPW + Afib → can accelerate accessory pathway → VF
🎯 Clinical Uses
Class II: AF/AFL rate control, SVT, post-MI, HFrEFClass IV: AF/AFL rate control, SVT termination (verapamil IV), HTNEsmolol: perioperative/ICU rate control
⭐ High-Yield Facts (Exam Focus)
Class II + Class IV = COMPLETE HEART BLOCK risk. NEVER combine.
Verapamil ↑ digoxin levels via P-gp inhibition — monitor if co-prescribed.
Non-DHP CCBs WORSEN HFrEF — avoid in systolic dysfunction.
Non-DHP CCBs CONTRAINDICATED in WPW + Afib (can accelerate accessory pathway → VF).
Esmolol: ideal perioperative/ICU rate control — t½ 9 min, rapidly titratable.
Verapamil + β-blocker = AV block. Verapamil + digoxin = digoxin toxicity. WPW contraindications: digoxin, verapamil, diltiazem, adenosine (all AV nodal blockers → force conduction via accessory pathway → VF).
Adenosine · Atropine · Magnesium Sulfate Other Antiarrhythmic Agents — Unique Mechanisms Adenosine: IV rapid bolus (t½ <10 sec) · Atropine: IV · Magnesium: IV
⚙ MOA
Adenosine: Binds A1 GPCR → activates GIRK K⁺ channels → K⁺ efflux → AV node hyperpolarization → transient complete AV block → terminates SVT. t½ <10 seconds.

Atropine: Muscarinic antagonist → blocks vagal tone → ↑ HR (positive chronotrope).

Magnesium sulfate: Competes with Ca²⁺ at channels → membrane stabilization → torsades.
⚡ Side Effects & Contraindications
Adenosine:
Transient flushing, chest tightness, dyspnea, sinus arrest (all <30 sec — extremely brief)
• Methylxanthines (theophylline, caffeine) BLOCK adenosine → need higher dose
• Dipyridamole POTENTIATES adenosine → need lower dose

Contraindicated: WPW + Afib (can shunt all conduction via accessory → VF), Sick sinus syndrome, 2nd/3rd degree AV block (without pacemaker)
🎯 Clinical Uses
Adenosine: acute SVT termination (AVNRT/AVRT) ★Adenosine: diagnostic (reveal flutter waves in Afib/AFL)Atropine: acute symptomatic bradycardia (1st line)Magnesium sulfate: Torsades de Pointes (1st line) ★
⭐ High-Yield Facts (Exam Focus)
Adenosine: drug of choice for acute SVT termination. t½ <10 SECONDS — effects vanish instantly.
Adenosine is both diagnostic + therapeutic: slowing AV node reveals flutter waves (sawtooth) in Afib/AFL.
Theophylline/caffeine BLOCK adenosine (competitive) → need higher dose.
Dipyridamole POTENTIATES adenosine (blocks reuptake) → need lower dose.
Magnesium sulfate: FIRST-LINE for Torsades de Pointes (IV, regardless of Mg²⁺ level).
Atropine: first-line for acute symptomatic bradycardia.
Adenosine in WPW + Afib = POTENTIALLY FATAL (blocks AV node → all impulses via accessory pathway → VF). Torsades = IV Mg²⁺ first (NOT Class IA or III — they worsen it by prolonging QT further).
Digoxin Cardiac Glycoside — Rate Control + Inotropy PO or IV · Narrow TI · Target 0.5–0.9 ng/mL · Monitor K⁺, Mg²⁺, Cr · Renally cleared
⚙ MOA
Inhibits Na⁺/K⁺-ATPase → ↑ intracellular Na⁺ → reverse NCX → ↑ intracellular Ca²⁺ → positive inotropy.
Also vagomimetic → ↑ vagal tone → ↓ AV node conduction → rate control in Afib.

Does NOT affect accessory pathway → dangerous in WPW.
⚡ Side Effects & Contraindications
Toxicity triad:
GI: anorexia, nausea, vomiting (earliest)
CNS: Xanthopsia (yellow-green visual halos)
Cardiac: PAT with block (classic!), bidirectional VT, any arrhythmia

Toxicity precipitated by:
• Hypokalemia (most important — K⁺ competes at binding site)
• Renal failure, hypomagnesemia, hypercalcemia
• Verapamil/amiodarone → ↑ digoxin levels (P-gp inhibition)

Treatment: DIGIBIND (Fab antibody fragments)
🎯 Clinical Uses
Afib rate control (esp. with HF)HFrEF symptom control (no mortality benefit)

Contraindicated in WPW + Afib.
⭐ High-Yield Facts (Exam Focus)
Classic toxicity triad: N/V + xanthopsia (yellow-green halos) + PAT with block.
Hypokalemia = most important precipitant (loop diuretics → hypoK⁺ → digoxin toxicity).
Verapamil and amiodarone → ↑ digoxin levels (P-gp inhibition) → toxicity.
Toxicity treatment: DIGIBIND (anti-digoxin Fab antibodies).
CONTRAINDICATED in WPW + Afib (AV nodal block → all impulses via accessory → VF).
Digoxin toxicity — check K⁺ FIRST. Hypokalemia is most common precipitant. Target level 0.5–0.9 ng/mL (NOT 1.0–2.0). PAT with block = classic digoxin toxicity arrhythmia. NEVER give in WPW + Afib.

↑ QRS Duration Na⁺ channel block (slows phase 0) Class IC (most) > IA > IB (minimal)
↑ PR Interval ↓ AV nodal conduction Class IC, II (β-block), IV (non-DHP CCB)
↑ QT Interval K⁺ channel block → delayed repolarization (↑ APD) Class IA, Class III — ALL → Torsades risk
↓ APD / minimal QT change Fast Na⁺ kinetics → rapid repolarization Class IB (lidocaine, mexiletine) — distinguishing feature
Torsades de Pointes QT prolongation + EADs (early afterdepolarizations) Rx: IV Magnesium sulfate 1st line + correct K⁺/Mg²⁺ + ↑ HR
WPW + Afib (irregular wide-complex tach) Accessory pathway bypass of AV node ⚠ Rx: PROCAINAMIDE. NEVER: adenosine, digoxin, verapamil, diltiazem
SVT (narrow complex tachycardia) Re-entry at AV node (AVNRT/AVRT) Rx: Adenosine (1st line). Verapamil IV (2nd). Terminates → AVNRT.
Pulseless VT/VF Life-threatening ventricular arrhythmia Rx: ★ Amiodarone 300 mg IV (ACLS 1st line). Alt: lidocaine.
